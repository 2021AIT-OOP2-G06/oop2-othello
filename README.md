# Requirement

* click==8.0.3
* Flask==2.0.2
* gunicorn==20.1.0
* itsdangerous==2.0.1
* Jinja2==3.0.3
* MarkupSafe==2.0.1
* numpy==1.22.1
* Werkzeug==2.0.2

# Installation
```
pip install -r requirements.txt
```
# Usage

```
git clone https://github.com/2021AIT-OOP2-G06/oop2_kadai11.git
cd oop2-othello
python main.py
```
